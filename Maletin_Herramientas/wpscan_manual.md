Comando / Opción	Qué hace
--url <URL>	Objetivo básico.
-e u	Enumeración de Usuarios: Intenta obtener la lista de usuarios registrados.
-e vp,vt	Busca específicamente plugins y temas vulnerables.
-P <archivo>	Lanza un ataque de fuerza bruta de contraseñas usando un diccionario.
--api-token <token>	RECOMENDADO: Usa el token gratuito de WPScan para mostrar datos detallados de vulnerabilidades (CVE).
--stealthy	Intenta evadir sistemas de detección con chequeos limitados.

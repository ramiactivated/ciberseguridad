whatweb <URL>	Escaneo básico: Muestra una línea resumen con el servidor, CMS y tecnologías detectadas.
-v <URL>	Modo Verbose: Muestra detalles completos de cada plugin y tecnología encontrada.
-a <nivel> <URL>	Nivel de Agresión: 1 (Pasivo), 3 (Agresivo), 4 (Pesado). El nivel 3 detecta versiones exactas de plugins.
-i <archivo>	Escanea múltiples objetivos desde un archivo de texto.
--proxy <IP:puerto>	Enruta el tráfico a través de un servidor proxy (como Tor o Burp).
--log-json=<ruta>	Guarda los resultados en formato JSON para análisis posterior.
